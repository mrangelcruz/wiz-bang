# Python Scripts

Python scripts for Wiz API automation and management tasks.

## Setup

```bash
cd scripts/python
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Authentication

Scripts use the Wiz API. Set environment variables:

```bash
export WIZ_CLIENT_ID="your-client-id"
export WIZ_CLIENT_SECRET="your-client-secret"
```
